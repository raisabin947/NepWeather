# 🌤️ NepWeather

NepWeather is a modern weather forecast web application built using HTML, CSS, and JavaScript. It provides real-time weather information, hourly forecasts, 5-day forecasts, air quality data, and dynamic weather backgrounds.

## ✨ Features

* 🌍 Search weather by city name
* 📍 Current location using Geolocation API
* 🌡️ Celsius/Fahrenheit toggle
* ⏰ Live date and time
* 🕒 Hourly weather forecast
* 📅 5-day weather forecast
* 🌬️ Air Quality Index (AQI)
* ☀️ UV Index support
* 🎨 Dynamic backgrounds based on weather conditions
* 💾 Saves the last searched city using Local Storage
* 📱 Fully responsive design
* ♿ Accessible and keyboard-friendly interface
* ❌ Error handling for invalid cities and network issues

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Fetch API
* OpenWeatherMap API
* Geolocation API
* Local Storage

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/nepweather.git
cd nepweather
```

### 2. Add Your API Key

Create a file named `config.js` and add:

```javascript
const API_KEY = "YOUR_OPENWEATHERMAP_API_KEY";
```

### 3. Run the Project

Open `index.html` in your browser or use a local server such as VS Code Live Server.

## 🔑 API

This project uses the OpenWeatherMap API.

* Current Weather API
* 5-Day Forecast API
* Air Pollution API
* One Call API (for UV Index)
